# Лабораторная №1 — BookShelf (CLI)

## Как запустить

1. Откройте проект **Command Line Tool** в Xcode.
2. Нажмите **Run**.
3. Вводите команды в консоли Xcode.

Команды
    •    help — показать справку
    •    add <title> | <author> | <year> | <genre> | <tags> — добавить книгу
    •    genre — одно из значений перечисления Genre
    •    tags — список тегов через запятую (можно пусто)
    •    list — вывести все книги (в читаемом виде)
    •    get <uuid> — вывести одну книгу по id
    •    search title <text> — поиск по названию
    •    search author <text> — поиск по автору
(поиск реализован минимум по 2 критериям)  ￼
    •    edit <uuid> [title="..."] [author="..."] [year=YYYY] [genre=...] [tags="t1,t2"] — редактировать книгу по id
    •    del <uuid> — удалить книгу по id
    •    exit — выход из программы

Дополнительные задания

Сделаны:
    •    D1. Валидация ввода:
title / author не пустые; publicationYear в разумном диапазоне; нормализация тегов (trim / регистр).  ￼
    •    D2. Ошибки как enum + понятные сообщения:
LibraryError/BookShelfError: Error с кейсами (некорректный ввод, не найдено, и т.д.) + человекочитаемые сообщения.  ￼
    •    D3. Редактирование книги:
команда изменения по id (title/author/year/genre/tags).  ￼
